# 行为准则

SaveAny 社区希望保持专业、友善、清晰和以问题为中心的协作氛围。参与 issue、PR、代码评审、文档讨论或安全报告时，请遵守以下准则。

## 我们鼓励

- 用事实、复现步骤和代码证据讨论问题。
- 尊重不同经验背景的贡献者。
- 对设计取舍提出清楚的理由。
- 在安全、版权、隐私和平台规则相关话题上保持谨慎。
- 接受维护者为了项目边界、法律风险或安全风险而拒绝某些能力。

## 我们不接受

- 人身攻击、骚扰、歧视、威胁或侮辱。
- 发布他人的隐私信息、密钥、Cookie、账号信息或未授权内容。
- 鼓励绕过 DRM、付费内容、私密内容、验证码、平台风控或登录限制。
- 在公开讨论中披露尚未修复的安全漏洞利用细节。
- 持续偏离主题、刷屏或恶意消耗维护者时间。

## 执行方式

维护者可以编辑、隐藏或删除不符合准则的内容，也可以临时或永久限制相关账号继续参与。安全漏洞、滥用和法律风险相关问题会优先处理。
